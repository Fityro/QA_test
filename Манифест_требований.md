# Training Catalog Examiner - Манифест требований

## Описание
VS Code расширение для выбора учебных направлений и запуска экзамена с интеграцией GitHub Copilot.

## Архитектура решения

### 1. Пользовательский интерфейс
- **Activity Bar Panel**: Собственная иконка "▶" в левой панели VS Code
- **Tree View**: Иерархическое дерево курсов с чекбоксами
- **Визуальные чекбоксы**: ☐ (не выбрано) / ☑️ (выбрано)
- **Кнопка запуска**: "▶" в заголовке панели для начала экзамена

### 2. Структура данных
- **Источник данных**: Папка `Database/Courses/` в workspace
- **Поддержка курсов**: 15 предустановленных направлений
- **Иерархия**: Курс → Модули → Уроки (автоматическое чтение из файловой системы)

### 3. Логика работы
#### Выбор курсов:
- Множественный выбор через чекбоксы
- Клик по элементу переключает состояние
- Визуальная обратная связь (эмодзи чекбоксы)

#### Запуск экзамена:
- Проверка наличия выбранных курсов
- Сохранение выбора в `user-selection.txt`
- Вывод в Output Channel для Copilot
- Уведомление пользователя

### 4. Интеграция с GitHub Copilot
- **Автоматическое чтение**: Copilot читает файл `user-selection.txt`
- **Источник контента**: Только материалы из `Database/` (никаких внешних источников)
- **Формат данных**: Структурированный текст с временными метками

## Технические требования

### Удобство запуска
- ✅ Горячая клавиша: `Ctrl+Alt+E`
- ✅ Иконка в Activity Bar
- ❌ Кнопка в статус-баре (убрана по требованию)
- ❌ Командная палитра (дополнительно)

### Отображение дерева
- ✅ Только папки с содержимым имеют стрелочки раскрытия
- ✅ Пустые папки отображаются как обычные элементы
- ✅ Чекбоксы у всех кликабельных элементов
- ✅ Правильное обновление состояния без дублирования

### Обработка ошибок
- ✅ Проверка существования папок Database
- ✅ Graceful handling пустых папок
- ✅ Валидация выбора (минимум 1 курс)

## Принципы работы

### Что РАБОТАЕТ:
- Простые эмодзи чекбоксы вместо сложных API
- Прямое обновление элементов дерева
- Чтение реальной структуры папок из Database
- Автоматическое определение пустых папок
- Сохранение в файл для интеграции с Copilot

### Что НЕ РАБОТАЕТ (избегать):
- VS Code TreeItemCheckboxState API (не поддерживается)
- Автоматическая отправка в чат (API ограничения)
- Буфер обмена для передачи данных
- Сложные Tree View провайдеры с множественным состоянием
- Дублирование элементов при обновлении

QA_test/

## Команды расширения
- `trainingCatalogExaminer.start` - Запуск через QuickPick (legacy)
- `trainingCatalogExaminer.toggleCourse` - Переключение чекбокса
- `trainingCatalogExaminer.startFromSelection` - Запуск экзамена

## Горячие клавиши
- `Ctrl+Alt+E` - Быстрый запуск выбора курсов
